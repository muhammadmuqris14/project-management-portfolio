## How These Tools Connect

All six artefacts are designed to work together as a single integrated project management system — not six separate tools.

The MS Project schedule provides the baseline dates that feed into the task tracker and status report milestone section. The task tracker mirrors the WBS structure from the MS Project schedule, with Task IDs cross-referenced in the RAID log. The RAID log items are summarised in the status report Risks and Issues section each week. The Power BI dashboard visualises completion rates, milestone status, and risk exposure drawn from the task tracker and RAID log data.

This is how a real PMO team operates — every tool connected, every update reflected across the full reporting chain.
